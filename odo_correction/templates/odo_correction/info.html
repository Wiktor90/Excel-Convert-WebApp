Instruction

1. Required file format (extension).

Only Excel files with .xlsx, .xls format extension or .csv file can be imported into the application. Any other type of file format is ignored. If you try to upload a not operand format, an error message will be displayed.

2. Required file content

Each file imported into the application (in the allowed format) must contain the following columns:

  • VEHICLE_ID_FW
  • ODOMETER_FW
  • TRANSACTION_DATE_FW (preferred format: 21-06-20)
  • TRANSACTION_TIME_FW

3. Table Columns & Buttons descriptions on File-List.

Table Columns:
  • Country – shows current country selection from drop-dawn in upload section
  • File Name – shows name of uploaded file
  • Correct File / Download Excel / Delete File – include actions buttons
  • Status – gray mark – file is not corrected; green mark – file is corrected, ready to upload
Buttons:

Correct – green (active) – clicking on this button is running the correction script in back-end. Site is loading then. During loading any other post/request action are not recommended. When script finish work, site will refresh automatically and you can see that buttons is - gray (disabled). You can also find green mark in status column, which means that file is already corrected.

Download – clicking on this button downloading the file on your local machine. Be aware that if you click download before file correction – you will download the same file which was uploaded!

Delete – clicking on this button deleting current record from list and what is more important from the DataBase. Is highly recommended to delete items after correction to save space on the server.
